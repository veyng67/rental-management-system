Rental Management System

Проект для управления недвижимостью и арендаторами.

Возможности

1.просмотр недвижимости
2.добавление недвижимости
3.удаление недвижимости
4.просмотр арендаторов
5.поиск арендаторов
6.фильтрация арендаторов

Технологии

1.Next.js
2.TypeScript
3.Tailwind CSS

Запуск

npm install
npm run dev
